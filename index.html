Circuit API examples

Build amazing real-time apps using Circuit APIs. Extend your site with video conferences, screensharing, or rich text messaging. Or create a bot to enhance the Circuit experience by integrating with third party services.

JavaScript SDK

JavaScript client-side library

Best suited for client-side web application. APIs for messaging, WebRTC incl. screenshare and much more.

node.js SDK

node.js JavaScript library

Best suited for bots or server-side web application. Same JS API as the JS SDK, including events. No WebRTC support.

REST API

RESTful API

Best suited for bots or server-side web application. No WebRTC support. Webhooks for events coming soon.

Polymer

Polymer web components

Best suited for enhancing existing web appplication with extremely simple to use web components.
